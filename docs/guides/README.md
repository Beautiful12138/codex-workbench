# Workbench 用户指南

这些文档面向使用者理解 Workbench 的设计思想、协作边界、状态模型和日常使用方式。它们不是任务状态真源，也不替代 `AGENTS.md`、`docs/policies/`、`.agents/skills/` 或 CLI 的实际门禁。
